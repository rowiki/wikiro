This folder contains modules contributed by the Romanian community for the pywikipedia framework [1]. 

[1] http://pywikipediabot.sourceforge.net/

List of files
----------------

diacritics_moves.py - moves the pages with incorrect diacritics in titles to pages with correct diacritics
diacritics_redirect.py - can create redirects from pages with correct to pages with incorrect diacritics, or the other way around
diacritics_replace.py - replaces incorrect diacritics with correct ones according to rules decided at [[:ro:WP:CD]]
duplicate_category.py - moves the text from a category to another category. Does not move any pages
pagerename.py - can rename a page according to some simple rules
